Memory Game:

Our website hosts the memory game. The classic memory game consists in a matrix of playing cards facing down which the player has to click. When clicked, the selected card will face up and the player will have to remember what it looks like and find its pair in a similar manner. The game ends when all pairs of cards have been found.
